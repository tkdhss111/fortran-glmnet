## üéØ Compilation Cheat Sheet

### Development (Fast Compilation)
```bash
ifx -O2 glmnet.f glmnet_wrapper.f90 your_program.f90
```

### Production (Recommended)
```bash
ifx -O3 -xHost -fp-model precise glmnet.f glmnet_wrapper.f90 your_program.f90
```

### Maximum Performance
```bash
ifx -O3 -xHost -ipo -qmkl glmnet.f glmnet_wrapper.f90 your_program.f90
```

### Debug
```bash
ifx -g -O0 -check all -traceback glmnet.f glmnet_wrapper.f90 your_program.f90
```

### Parallel Processing
```bash
ifx -O3 -xHost -qopenmp glmnet.f glmnet_wrapper.f90 your_program.f90
export OMP_NUM_THREADS=8
```
